   项目简介: Headroom 是一个专为 AI Agent 设计的上下文压缩层，能够在不影响回答质量的前提下，将输入 LLM 的 token 数量减少 60%–95%。它通过本地运行的库、代理、MCP 服务器或中间件，智能压缩工具输出、日志、RAG 片段、文件内容和对话历史，并采用可逆压缩（CCR）机制确保原始数据可随时恢复。支持 Claude Code、Cursor、Codex、Aider、Copilot CLI 等主流编码 Agent，以及 LangChain、Agno、Strands、Vercel AI SDK 等主流框架。
   亮点: 核心功能：多算法上下文压缩（SmartCrusher JSON压缩、CodeCompressor AST感知代码压缩、Kompress-base 文本压缩），实现 60%-95% token 节省; 解决痛点：AI Agent 长上下文场景下 token 成本爆炸、上下文窗口溢出、多 Agent 记忆孤岛问题; 亮点3：可逆压缩（CCR）机制——原始数据本地存储，LLM 可按需检索，兼顾压缩效率与信息完整性
   详细报告: Headroom 是专为 AI Agent 打造的「上下文压缩层」，核心使命是：让 AI Agent 在读取海量上下文（工具输出、日志、RAG 片段、文件、对话历史）时，消耗更少 token，却能获得相同质量的回答。

**核心工作原理：**
Headroom 在 AI Agent 和 LLM 提供商之间插入一个本地运行的压缩管道。当 Agent 准备向 LLM 发送请求时，Headroom 会：
1. **ContentRouter（内容路由器）**：自动检测输入内容的类型（JSON、代码、纯文本、图片等），并路由到对应的压缩算法
2. **三大压缩引擎**：
   - **SmartCrusher**：针对 JSON 数据（如 API 响应、工具输出）进行智能压缩，处理嵌套对象、数组、混合类型
   - **CodeCompressor**：基于 AST（抽象语法树）感知的代码压缩，支持 Python、JavaScript、Go、Rust、Java、C++ 等语言，保留语义结构的同时大幅精简
   - **Kompress-base**：自研的 HuggingFace 模型，专门针对 Agent 交互痕迹训练的文本压缩模型
3. **CacheAligner（缓存对齐器）**：稳定化 prompt 前缀，使 Anthropic/OpenAI 的 KV 缓存命中率最大化，进一步降低重复计算成本
4. **CCR（可逆压缩）**：原始数据永不删除，存储在本地；当 LLM 需要查看原始内容时，可通过 `headroom_retrieve` 工具按需取回

**部署方式极其灵活：**
- **库模式**：`compress(messages)` 一行代码嵌入 Python/TypeScript 应用
- **代理模式**：`headroom proxy --port 8787` 零代码改动，拦截所有 LLM 请求
- **Agent 包装模式**：`headroom wrap claude|codex|cursor|aider|copilot` 一键包装主流编码 Agent
- **MCP 服务器**：提供 `headroom_compress`、`headroom_retrieve`、`headroom_stats` 工具，任何 MCP 客户端均可接入
- **中间件模式**：ASGI 中间件、LangChain 模型包装、Agno 模型包装、Vercel AI SDK 中间件等

**跨 Agent 记忆共享：**
Headroom 提供跨 Agent 的共享存储，支持 Claude、Codex、Gemini 等 Agent 之间的上下文共享和自动去重，解决多 Agent 协作时的记忆孤岛问题。

**headroom learn（失败学习）：**
自动挖掘失败的会话记录，将修正建议写入 `CLAUDE.md`、`AGENTS.md` 或 `GEMINI.md`，实现 Agent 的自我进化。

   项目简介: MiroFish 是一款基于多智能体（Multi-Agent）技术的下一代 AI 预测引擎，致力于通过构建高保真的平行数字世界来模拟现实世界的复杂演化。用户只需上传种子信息（如新闻、政策草案、金融信号或小说片段），并用自然语言描述预测需求，系统即可自动创建数千个具有独立人格、长期记忆和行为逻辑的智能体，在数字沙盒中进行社会交互与演化，最终生成详尽的预测报告和可深度交互的模拟世界。其核心理念是‘在数字沙盒中预演未来，在无数次模拟后赢得决策’，适用于舆情推演、政策测试、金融预测乃至文学创作等多元场景。

   详细报告: MiroFish 是一个极具前瞻性的群体智能预测平台，它不依赖传统统计模型，而是通过构建一个由 AI 智能体组成的‘平行世界’来模拟现实社会的复杂动态。其工作流程分为五个阶段：图构建（Graph Building）、环境搭建（Environment Setup）、模拟运行（Simulation）、报告生成（Report Generation）和深度交互（Deep Interaction）。首先，系统从真实世界的种子材料（如新闻报道、政策文本或小说内容）中提取关键实体与关系，利用 GraphRAG 技术构建知识图谱，并为每个智能体注入个性化记忆和行为规则。随后，在双平台并行模拟环境中，成千上万的 Agent 基于 LLM 驱动进行社交互动、信息传播和决策演化，系统自动解析用户预测需求并动态更新时空记忆。模拟结束后，ReportAgent 利用丰富的工具集对结果进行深度分析，生成结构化预测报告。更酷的是，用户可以在模拟结束后‘进入’这个数字世界，与任意 Agent 对话，探索不同变量下的未来走向。项目底层依托 CAMEL-AI 的 OASIS 框架，使用 Python + Node.js 全栈架构，支持 Docker 一键部署，并推荐使用阿里云百炼平台的 Qwen-plus 模型作为 LLM 后端。尽管目前 GitHub 指标显示项目尚处早期（无 star、fork 或近期提交），但其技术愿景宏大，已在武汉大学舆情模拟和《红楼梦》遗失结局推演等案例中展示了强大潜力。
